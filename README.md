Anomaly Detection at Lepton Collider. We used the FCCee IDEA detector configuration.

3. FCC Analyses: https://github.com/HEP-FCC/FCCAnalyses
4. ROOT TMVA
5. Scikit Learn
6. XGBoost
7. Matplotlib
8. Uproot, pandas, numpy
